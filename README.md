# Higher Level Programming - Holberton School

This repository contains programs written for the higher-level programming track at Holberton School.  
The projects in this repository are written mainly in **Python**, with some projects in other high-level languages.

## Contents
Each folder in this repository corresponds to a project with multiple tasks.  
Examples of topics covered include:

- Python basics
- Control flow, loops, and functions
- Data structures and algorithms
- Object-oriented programming
- File I/O in Python
- Test-driven development
- Higher-level concepts and tools

## Requirements
- All code runs on **Ubuntu 20.04 LTS**
- Python scripts use **Python 3.8**
- Code follows **pycodestyle** (PEP8) style guidelines
- Each project contains its own README explaining details
