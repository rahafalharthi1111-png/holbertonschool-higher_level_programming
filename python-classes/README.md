Python - Classes and Objects

This project introduces the fundamental concepts of Object-Oriented Programming (OOP) in Python. You will learn how to define classes, create objects, work with attributes, and apply basic principles of data encapsulation and abstraction.

Learning Objectives

By the end of this project, you should be able to explain:

General

What is OOP

“First-class everything”

What is a class

What is an object and an instance

The difference between a class and an instance

What is an attribute

How to use public, protected, and private attributes

What is self

What is a method

What is the __init__ method

What is Data Abstraction, Data Encapsulation, and Information Hiding

What is a property

The difference between an attribute and a property

How to write Pythonic getters and setters

How to dynamically create attributes for instances

How to bind attributes to classes and objects

What __dict__ contains

How Python performs attribute lookup

How to use getattr()

Requirements
General

Allowed editors: vi, vim, emacs

All files interpreted/compiled on Ubuntu 20.04 LTS using Python 3.8.5

All files should end with a new line

The first line of every file must be: #!/usr/bin/python3

Code must follow pycodestyle (version 2.7.*)

All files must be executable

File lengths will be tested using wc

Documentation

Each module, class, and method must contain proper documentation following the Google Python Style Guide. The documentation must explain the purpose of the module, class, or method.
