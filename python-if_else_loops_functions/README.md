# Python - More Control Flow

This project covers the fundamentals of Python control flow, including conditions, loops, functions, variable scope, and error handling.

## Learning Objectives
- Understand Python indentation
- Use `if`, `elif`, and `else` statements
- Use comments and assign variables
- Work with `while` and `for` loops
- Use `break` and `continue`
- Understand `else` in loops
- Understand when to use the `pass` statement
- Use the `range()` function
- Write and call functions
- Understand function return values
- Learn variable scope
- Read and understand Python tracebacks
- Use Python arithmetic operators
